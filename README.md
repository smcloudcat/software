# 小猫咪软件库

#### 介绍
软件分享界面（应用商城），带后台，可以上传文件，分享文件


#### 安装教程

1.  上传到服务器
2.  安装程序：域名/install.php
3.  默认后台地址为admin
4.  账号密码：admin/123456

#### 使用说明

1.  自己摸索吧
2.  本程序后台建议用电脑操作，对手机用户并不友好
